## 🚀 Features

- 📋 Multi-platform Support: Windows, macOS
- 🔄 Smart Sync: Real-time clipboard content synchronization between devices
- 🔐 Secure Encryption: End-to-end encryption ensures data security
- 📱 Coming Soon: iOS and Android mobile apps, and Linux (Desktop)
- 🌐 Multiple Deployment Options: Local, Cloud, and Private Cloud editions

## 📦 Edition Details

### Local Edition
- Completely offline operation, local data storage
- Ideal for personal users and privacy-conscious scenarios

### Cloud Edition
- Based on public cloud services
- Real-time cross-device synchronization
- Suitable for individuals and small teams

### Enterprise Edition
- Supports private deployment
- Complete control over data storage and transmission
- Ideal for enterprises and organizations
- Professional deployment support

## 💻 Supported Platforms

### Currently Supported
- Windows (7/8/10/11)
- macOS (10.14+)

### Coming Soon
- iOS (iOS 13.0+)
- Android (Android 8.0+)
- Linux (Major distributions)
