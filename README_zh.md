## 🚀 特性

- 📋 多平台支持：Windows、macOS
- 🔄 智能同步：设备间实时同步剪贴板内容
- 🔐 安全加密：端到端加密确保数据安全
- 📱 即将支持：iOS 和 Android 移动端以及Linux（桌面版）
- 🌐 多版本部署：离线版、云端版、私有云版

## 📦 版本说明

### 离线版 (Local Edition)
- 完全离线运行，数据存储在本地
- 适合个人用户和注重隐私的场景

### 云端版 (Cloud Edition)
- 基于公有云服务
- 跨设备实时同步
- 适合个人和小型团队

### 私有云版 (Enterprise Edition)
- 支持私有化部署
- 完全控制数据存储和传输
- 适合企业和组织使用
- 提供专业的部署支持

## 💻 支持平台

### 当前支持
- Windows (7/8/10/11)
- macOS (10.14+)

### 即将支持
- iOS (iOS 13.0+)
- Android (Android 8.0+)
- Linux (主流发行版)

- 自定义规则
- 智能分类

## 🔐 安全性

## 📊 系统要求
### Windows
- Windows 7 及以上版本
- .NET Framework 4.6.1 及以上
- 8GB 内存及以上
- 200MB 硬盘空间

### macOS
- macOS 10.14 及以上版本
- 4GB 内存及以上
- 200MB 硬盘空间
