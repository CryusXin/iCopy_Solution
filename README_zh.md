## 🚀 特性

- 📋 多平台支持：Windows、macOS、Linux（桌面版）
- 🔄 智能同步：设备间实时同步剪贴板内容
- 🔐 安全加密：端到端加密确保数据安全
- 📱 即将支持：iOS 和 Android 移动端
- 🌐 多版本部署：本地版、公有云版、私有云部署版

## 📦 版本说明

### 本地版 (Local Edition)
- 完全离线运行，数据存储在本地
- 适合个人用户和注重隐私的场景
- 支持本地设备间同步

### 公有云版 (Cloud Edition)
- 基于公有云服务
- 跨设备实时同步
- 自动备份与恢复
- 适合个人和小型团队

### 私有云部署版 (Enterprise Edition)
- 支持私有化部署
- 完全控制数据存储和传输
- 适合企业和组织使用
- 提供专业的部署支持

## 💻 支持平台

### 当前支持
- Windows (7/8/10/11)
- macOS (10.14+)
- Linux (主流发行版)

### 即将支持
- iOS (iOS 13.0+)
- Android (Android 8.0+)
